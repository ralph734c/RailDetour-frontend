# RailDetour

A Fullstack Web Application focused on alerting folks if a train is crossing nearby and providing detours.

This repo contains the frontend of this project.

## Tech Stack

**Frontend:**
- JavaScript
- React
- Recoil
- Vite
- TailwindCSS
- Supabase for Auth
- Axios for communication with the backend
